❓ 핵심 Q&A
Q 클로드 프로 vs 맥스 — 어떤 플랜을 써야 하나?
프로는 Opus 사용 시 5시간 한도로 금방 소진. 맥스 x5가 중소기업 수준. 낮은 플랜으로 시작 후 업그레이드 권장.
Q 튜브워커 신청·수령 방법은?
스킬로 만든 대본을 카페 과제제출 게시판에 업로드하면, 이메일로 프로그램 접속 정보 발송.
Q 스킬 제작은 Claude Code에서? 일반 Claude.ai에서?
일반 Claude.ai에서 진행하고, 완성된 스킬 파일을 Claude Code에 설치만 하는 방식이 토큰 효율이 높음.

❓ Q&A

Q
유튜브 대본(자막) 추출은 어떻게 하나요? 벤치마킹 채널 분석 방법은?
12회
경험자

방법 1) grabbit 크롬 확장프로그램으로 자막 스크래핑. 방법 2) NotebookLM에 URL 넣어 자막 추출. 방법 3) 잼스 제작 IVT_Pickup 도구로 채널 단위 일괄 추출. 방법 4) 바이브코딩으로 직접 도구 제작.

Q
강의 진도를 못따라가서 초조한데 어떻게 해야 하나요?
8회
강사

정규 라이브 강의만 우선 수강하면 충분. 프리미엄반은 플러스알파이므로 기본만 제대로 하면 모든 것이 가능. 각자의 시간대와 페이스에 맞춰 진행.

Q
유튜브 채널 카테고리(경제학 등) 경쟁이 치열한데 어떻게 차별화하나요?
7회
경험자

카테고리를 니치로 세분화하여 진입. 경제학이라면 금, 은, 주식, 거시경제 등으로 세분화. 뉴스 단순 스크래핑보다 고유한 결(톤)과 독자적 각도를 잡아야. 롱폼 업로드 후 2주 정도 추이 관찰.

Q
VSCode에서 bypass permissions이 안 되는데 어떻게?
4회
경험자

VSCode 확장프로그램 클로드코드 설치 후 설정 화면에서 바이패스 옵션 활성화. 가든 님이 작성한 PDF 가이드 참고.

Q
유튜브 계정 분리와 애드센스 매칭은 어떻게?
3회
경험자

기본 원칙은 1계정 1채널 1애드센스. 여러 가지 편법 존재. 유랑 님의 유튜브 채널 영상 참고 권장.

Q
autoworkers-1st.zip과 autoworkers-1st-skill.zip은 동일한 파일인가요?
3회
경험자

첫 번째 배포본은 채널명이 한국어로 되어 오류가 발생. 두 번째 배포본은 영어로 변경된 버전. 다운로드 후 압축 해제하여 클로드에서 폴더 경로 복사 후 설치 요청.

Q
클로드는 유튜브 영상 분석이 안 되는데, 제미나이처럼 가능하지 않나?
2회
강사

클로드는 유튜브 영상 직접 분석 불가. 클로드는 파이썬 코드를 이용해 더 정확하게 분석. 목요일 강의에서 URL만 주면 분석-대본 생성하는 스킬 배울 예정.

Q
클로드 메모리가 최신화했는데 계속 예전 내용을 사용합니다.
2회
미해결

에이전트와 스킬이 건너뛰며 이전 폼으로 결과물이 나오는 문제. 작업 지시를 강제하는 방법과 폼 재구성 시도가 언급됨.

Q
구글 클라우드(버텍스AI) 무료 크레딧 $300은 어떻게 받나?
2회
경험자

버텍스AI로 가입하면 구글 클라우드에서 90일간 $300 크레딧 제공. 버텍스에서 API를 받아 연결해야 크레딧으로 사용 가능.
